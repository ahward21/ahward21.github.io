Enschede Museum Nacht

Museumnacht Feedback System

About this project

The task was to create a feedback system for Museum Nacht Enschede, the event is a showcase of Enschede's local and student gems. Over 33 locations around Enschede city center are lit up for one night a year. To gage both visitors opinions and experiences a member of the organisation hired me to create this system for them.
