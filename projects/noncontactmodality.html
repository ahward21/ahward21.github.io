Non-Contact Modality Feedback

Utilizing non-contact feedback within Virtual Reality environments

About this project

This research project explores innovative ways to provide haptic and sensory feedback in VR environments without requiring physical contact with devices. Using Heat towers, hair dryers and humdifiers to create heat, and wind, we were able to create a system that could provide feedback to the user in a non-contact manner. I was brough on to help apply VR to the system, using a boat in Unity and physical rowing mechanics. Unity triggers certain towers, hair dryers based on location of the VR player and their boat.

Project Details

Technologies

Unity, C#,Blender, Arduino, Custom Hardware

Duration

1 month

Role

Developer

Results

The main goal was to create a system that could provide feedback to the user in a non-contact manner and to find a way to test the system. I was able to apply VR to the system for participants to test the system. The project supervisor is a Phd candidate in HCI - Marie-Laure
